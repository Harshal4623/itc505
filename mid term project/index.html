Story Planning

I planned the game as a branching story with multiple paths leading to eight unique endings. The player begins at the Moonlit Palace entrance and must choose between entering the palace or exploring the moon garden. Each choice leads to another scene with new decisions, and those decisions eventually lead to one of the final endings.

Project Addendum

For my mid-term project, I created an interactive “Choose Your Own Adventure” game titled Moonlit Palace Adventure. The purpose of this project was to apply HTML, CSS, and JavaScript in a creative way while building an interactive webpage. Before coding, I planned the story structure carefully so that the player would have multiple branching choices leading to different outcomes. I wanted the story to feel magical and different from a typical adventure game, so I chose a moonlit palace theme with mystery, royalty, hidden rooms, and peaceful gardens.

I built the webpage using separate HTML, CSS, and JavaScript files. The HTML provides the main structure of the page, including the story area, buttons, image section, addendum, and footer. CSS was used to create an aesthetic design with dark blue and purple gradients, glowing accents, rounded cards, and animated decorative effects such as stars and floating light glows. I also added transitions and hover effects to make the page feel more polished and visually engaging.

JavaScript controls the interactive behavior of the game. I used an object-based structure to store each scene’s text, image, and choices. Event listeners are attached to buttons so that when the player clicks a choice, the story, choices, and image update dynamically without reloading the page. One challenge was making sure that every path worked correctly and that all eight endings were reachable. I solved this by organizing the scene names clearly and testing each branch step by step. Through this project, I improved my understanding of DOM manipulation, event handling, debugging, and how creativity can be combined with programming to build an engaging user interface.
